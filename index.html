  • Quiz Game

    This is a quiz game I built using HTML, CSS, vanilla Javascript and the Open Trivia Database API. Async function calls API fetch request based on a switch statement that includes 8 possible quiz categories that the user selects. Then, a for loop randomizes the order of the possible answers that are displayed for each question, and class names are added and removed to indicate correct/incorrect answers.

  • RGB to Hex Converter

    This page uses a function that takes any RGB value and results in the hexidecimal representation being returned.
    The RGB color code values are based on the Base-10 decimal number system (0-9). The valid RGB values are numbers 0-255. The Hexadecimal number system is a Base-16 system (0-9 followed by the letters A-F).
    This is just a fun little conversion from RGB to Hex.
